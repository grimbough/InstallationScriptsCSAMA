Course materials

You can find curated course materials here.
Any course-specific packages can be installed from the local repository (see below).

You can also find the course Github repository at https://github.com/Bioconductor/CSAMA/

Initial setup script

Run the following command in your R session to install all the packages required for the course.

source("http://192.168.0.5/install_packages.R")

Using the local R package repository

First you need to set up the local mirrors:

options(repos = "http://192.168.0.5/packages/cran") options(BioC_mirror = "http://192.168.0.5/")

Packages can be then installed using:

BiocManager::install("package_name")
