Course materials

You can find the course materials here.
The course-specific packages can be installed from the local repository (see below).

Initial setup script

Run the following command in your R session to install all the packages required for the course.

source("http://192.168.0.9/install_packages.R")

Using the local R package repository

First you need to set up the local mirrors:

source("http://192.168.0.9/biocLite.R")

Packages can be then installed using:

biocLite("package_name")

We also have lab specific install scripts if you are missing packages for a single lab. You only need to run the appropriate line below for the lab you are missing

source("http://192.168.0.9/lab1_packages.R")

source("http://192.168.0.9/lab3_packages.R")

source("http://192.168.0.9/lab4_packages.R")

source("http://192.168.0.9/lab5_packages.R")

source("http://192.168.0.9/lab6+7_packages.R")

source("http://192.168.0.9/lab8_packages.R")
